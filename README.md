## 🎮 How to Use

-   **👆 Counting Time**: Just smash that "INCREMENT" button to see the numbers rise. Simple, right?

### Code Snippet 📝

```javascript
let countEl = document.getElementById("count-el");
let count = 0;

function increment() {
	count += 1;
	countEl.innerText = count;
}
```
